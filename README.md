# TypeTime

A minimalist typing speed test app built with React and Vite. Test your WPM, track accuracy, and switch between multiple themes.

## Features

- Real-time WPM and accuracy tracking
- Multiple themes: Dark, Light, Cyberpunk, Matrix
- Clean, distraction-free interface

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- npm (comes with Node.js)

### Installation

```bash
git clone https://github.com/Farhan2560/TypeTime.git
cd TypeTime
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

The output will be in the `dist/` folder.

### Preview the Production Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```
